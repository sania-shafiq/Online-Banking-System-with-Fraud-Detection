# 🏦 Online Banking System with Fraud Detection

## 📌 Overview

This is a web-based **Online Banking System** developed using **PHP (Backend)** and **XAMPP (Apache & MySQL)**. It allows users to securely manage accounts, perform deposits and withdrawals, and track transaction history. A built-in fraud detection module monitors and flags suspicious transactions.

---

## 🚀 Features

* User registration & login (password hashing)
* Deposit & withdraw funds
* Real-time balance updates
* Transaction history
* Rule-based fraud detection
* Fraud alert logging

---

## 🛠 Technologies

* PHP
* MySQL
* HTML, CSS, JavaScript
* XAMPP (Apache Server)
